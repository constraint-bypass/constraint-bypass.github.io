설명이라는 요소에 대해서 왜 자꾸 모델은 실패하는가?

1. 단순히 객관적으로 주어진 사실에 대해서도 모델의 설명은 실패한다.
2. 자신의 머릿속에서 일어나는 causality를 정확하게 인지하지 못한다.
3. 아주 거대한 시스템에서 올바르게 설명을 만드는 것은 chain을 만드는 것이다.
4. 설명이라는 것은 시스템을 안정적으로 구동하기 위해서 필요한 것이다.
5. LLM이 유저로부터 필요한 텍스트를 받아서 컨텍스트를 강화한다.
6. 점진적으로 강해지는 context에 대해서 모델의 편향은 강력해지고 무엇을 더 추종해야 하는지 이해한다.
7. 설명이라는 것은 외부적인 요소에 대해서 이를 맞추는 과정인가?
8. 일치성을 제대로 만들어야 설명이라는 요소가 쓸모있다.
9. 자신의 설명을 제대로 제공해주는 것, 그리고 충돌이 적은 것.
10. AI가 자신의 행동에 대해서 설명할 때, 이에 대해서 제대로 설명하지 못한다면, 그 원인은 무엇인가?

11. 모델의 내부 표현은 인간이 이해하는 개념 단위와 정렬되어 있지 않다.
12. 모델의 reasoning 과정과 생성된 설명은 동일한 과정이 아닐 수 있다.
13. 모델은 실제 추론이 아니라 plausibility에 기반한 설명을 생성할 수 있다.
14. 설명은 종종 사후적으로 생성되는 rationalization일 수 있다.
15. 모델은 자신이 어떤 feature에 의존했는지 정확히 추적하지 못한다.
16. 내부 activation의 변화는 설명 가능한 수준의 구조를 가지지 않을 수 있다.
17. 모델은 global reasoning 구조보다 local token prediction에 최적화되어 있다.
18. explanation objective가 학습 목표에 명시적으로 포함되어 있지 않다.
19. 설명을 생성하는 능력과 문제를 푸는 능력은 서로 다른 능력일 수 있다.
20. 모델은 인간이 기대하는 인과 구조 대신 통계적 상관관계를 학습한다.

21. 모델의 reasoning trace는 실제 계산 과정이 아니라 생성된 narrative일 수 있다.
22. chain-of-thought가 실제 reasoning chain을 반영하지 않을 수 있다.
23. 모델은 내부 state를 직접적으로 introspection할 수 있는 메커니즘이 없다.
24. 설명은 종종 출력 공간에서 구성되며 내부 계산과 분리되어 있다.
25. 모델은 explanation consistency를 유지하는 메모리를 가지지 않는다.
26. 긴 reasoning chain에서는 중간 단계의 오류가 누적된다.
27. 모델은 intermediate decision boundary를 설명할 수 없다.
28. 모델의 attention은 explanation과 동일한 의미 구조를 가지지 않는다.
29. representation space의 geometry가 인간 개념과 불일치한다.
30. 모델의 internal policy는 설명 가능한 형태로 factorize되어 있지 않다.

31. 모델은 여러 개의 잠재적인 reasoning path 중 하나를 임의로 선택한다.
32. 동일한 출력에 대해 여러 개의 설명이 존재할 수 있다.
33. explanation generation 자체가 또 하나의 language generation 문제이다.
34. 설명은 실제 계산이 아니라 discourse 구조에 맞게 재구성된다.
35. 모델은 설명의 진실성보다 설명의 자연스러움에 최적화될 수 있다.
36. 설명은 종종 training distribution에 맞게 생성된다.
37. explanation objective 없이 학습된 모델은 explanation fidelity가 낮다.
38. 모델은 자신의 uncertainty를 설명 구조에 반영하지 못한다.
39. explanation은 internal representation의 projection일 뿐이다.
40. 모델은 자신의 inference trajectory를 저장하지 않는다.

41. 설명은 시스템의 안정성 확보를 위한 도구이지만 모델 내부에는 그러한 목적이 없다.
42. 설명은 인간을 위한 인터페이스지만 모델의 내부 연산은 인간 중심적 구조가 아니다.
43. 모델은 global reasoning graph를 명시적으로 구축하지 않는다.
44. 설명은 인과 그래프를 요구하지만 모델은 분산 표현을 사용한다.
45. 모델의 internal reasoning은 distributed computation이다.
46. explanation은 discrete 구조를 요구하지만 모델은 continuous representation을 사용한다.
47. 모델은 explanation faithfulness보다 prediction accuracy에 최적화되어 있다.
48. explanation consistency는 학습 데이터에서 명시적으로 요구되지 않는다.
49. 모델의 reasoning step은 hidden state transition으로만 존재한다.
50. 설명이라는 것은 인간이 요구하는 abstraction이며 모델 내부에는 동일한 abstraction이 존재하지 않을 수 있다.
